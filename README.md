# Social-Network-API

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)


## Description

 Backend API for a social network web application where users can share their thoughts, react to friendsâ€™ thoughts, and create a friend list. 
 

 Built with Express.js for routing, a MongoDB database, and the Mongoose ODM, & and Moment.js to format timestamps.

## Installation

The user should clone the repository from GitHub. 

     git clone https://github.com/GEEZEE91/E-Commerce-Back-End.git

Install dependencies

    npm i

After installations is completed, run the app with below in terminal

    npm start
    
    
Open MongoDB and create a connection

![mongo](https://user-images.githubusercontent.com/3950562/212976506-89112067-f8d1-43d5-9e06-b3e8bee988d8.png)



Open up Insomnia and test API end points for users and thoughts and create seed data.

![insomnia](https://user-images.githubusercontent.com/3950562/212976409-2e7d2886-0ded-4e48-a7aa-4d957c569a7b.png)
